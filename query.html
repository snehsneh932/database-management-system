SQL Queries and Concepts

What is a Table?

A table in a database is a collection of related data entries organized in rows and columns.

Use of Tables

  • Stores structured data
  • Allows efficient access and manipulation
  • Helps maintain relationships between different data sets

How to Create a Table (SQL Command)

CREATE TABLE Students (
  ID INT PRIMARY KEY,
  Name VARCHAR(100),
  Age INT,
  Grade VARCHAR(10)
);

Inserting Data into a Table (SQL Command)

INSERT INTO Students (ID, Name, Age, Grade)
VALUES (1, 'Alice', 20, 'A');

Selecting Data from a Table (SQL Command)

SELECT * FROM Students;

-- Or specific columns:
SELECT Name, Grade FROM Students;

Updating Data in a Table (SQL Command)

UPDATE Students
SET Grade = 'A+'
WHERE ID = 1;

Deleting Rows from a Table (SQL Command)

DELETE FROM Students
WHERE ID = 1;

Altering a Table (SQL Command)

-- Add a new column:
ALTER TABLE Students
ADD Email VARCHAR(100);

-- Drop a column:
ALTER TABLE Students
DROP COLUMN Email;

Important SQL Queries & What They Do

  • SELECT * – Retrieves all data from a table.
  • SELECT COUNT(*) – Counts number of records.
  • WHERE – Filters records based on conditions.
  • ORDER BY – Sorts results in ascending or descending order.
  • GROUP BY – Groups rows sharing a property for aggregate functions.
  • JOIN – Combines rows from two or more tables.
  • LIKE – Searches for a specified pattern.
  • IN – Checks if a value matches any in a list.
  • BETWEEN – Filters values within a range.
